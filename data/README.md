# Dataset

Primary Dataset:
https://nasa-public-data.s3.amazonaws.com/plot3d_utilities/dataset-processed.zip

Dataset contains:
- Airfoil geometry
- Angle of Attack (AoA)
- Reynolds Number
- Lift Coefficient (Cl)
- Drag Coefficient (Cd)
- Pressure Coefficient (Cp)

Dataset files are not tracked by git.
Download manually and place contents in this directory.
